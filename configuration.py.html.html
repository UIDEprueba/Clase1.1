41   <span class="s0">class </span><span class="s1">ConfigurationCommand(Command):</span>
42       <span class="s2">&quot;&quot;&quot; 
43       Manage local and global configuration. 
44    
45       Subcommands: 
46    
47       - list: List the active configuration (or from the file specified) 
48       - edit: Edit the configuration file in an editor 
49       - get: Get the value associated with name 
50       - set: Set the name=value 
51       - unset: Unset the value associated with name 
52       - debug: List the configuration files and values defined under them 
53    
54       If none of --user, --global and --site are passed, a virtual 
55       environment configuration file is used if one is active and the file 
56       exists. Otherwise, all modifications happen to the user file by 
57       default.
