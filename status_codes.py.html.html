20   <pre><span class="s0">r&quot;&quot;&quot; 
21   The ``codes`` object defines a mapping from common names for HTTP statuses 
22   to their numerical codes, accessible either as attributes or as dictionary 
23   items. 
24    
25   Example:: 
26    
27       &gt;&gt;&gt; import requests 
28       &gt;&gt;&gt; requests.codes['temporary_redirect'] 
29       307 
30       &gt;&gt;&gt; requests.codes.teapot 
31       418 
32       &gt;&gt;&gt; requests.codes['\o/'] 
33       200 
34    
35   Some codes have multiple names, and both upper- and lower-case versions of 
36   the names are allowed. For example, ``codes.ok``, ``codes.OK``, and 
37   ``codes.okay`` all correspond to the HTTP status code 200.
