401  <span class="s0">class </span><span class="s1">LocalProxy(t.Generic[T]):</span>
402      <span class="s3">&quot;&quot;&quot;A proxy to the object bound to a context-local object. All 
403      operations on the proxy are forwarded to the bound object. If no 
404      object is bound, a ``RuntimeError`` is raised. 
405   
406      :param local: The context-local object that provides the proxied 
407          object. 
408      :param name: Proxy this attribute from the proxied object. 
409      :param unbound_message: The error message to show if the 
410          context-local object is unbound. 
411   
412      Proxy a :class:`~contextvars.ContextVar` to make it easier to 
413      access. Pass a name to proxy that attribute. 
414   
415      .. code-block:: python 
416   
417          _request_var = ContextVar(&quot;request&quot;) 
418          request = LocalProxy(_request_var) 
419          session = LocalProxy(_request_var, &quot;session&quot;) 
420   
421      Proxy an attribute on a :class:`Local` namespace by calling the 
422      local with the attribute name: 
423   
424      .. code-block:: python 
425   
426          data = Local() 
427          user = data(&quot;user&quot;) 
428   
429      Proxy the top item on a :class:`LocalStack` by calling the local. 
430      Pass a name to proxy that attribute. 
431   
432      .. code-block:: 
433   
434          app_stack = LocalStack() 
435          current_app = app_stack() 
436          g = app_stack(&quot;g&quot;) 
437   
438      Pass a function to proxy the return value from that function. This 
439      was previously used to access attributes of local objects before 
440      that was supported directly. 
441   
442      .. code-block:: python 
443   
444          session = LocalProxy(lambda: request.session) 
445   
446      ``__repr__`` and ``__class__`` are proxied, so ``repr(x)`` and 
447      ``isinstance(x, cls)`` will look like the proxied object. Use 
448      ``issubclass(type(x), LocalProxy)`` to check if an object is a 
449      proxy. 
450   
451      .. code-block:: python 
452   
453          repr(user)  # &lt;User admin&gt; 
454          isinstance(user, User)  # True 
455          issubclass(type(user), LocalProxy)  # True 
456   
457      .. versionchanged:: 2.2.2 
458          ``__wrapped__`` is set when wrapping an object, not only when 
459          wrapping a function, to prevent doctest from failing. 
460   
461      .. versionchanged:: 2.2 
462          Can proxy a ``ContextVar`` or ``LocalStack`` directly. 
463   
464      .. versionchanged:: 2.2 
465          The ``name`` parameter can be used with any proxied object, not 
466          only ``Local``. 
467   
468      .. versionchanged:: 2.2 
469          Added the ``unbound_message`` parameter. 
470   
471      .. versionchanged:: 2.0 
472          Updated proxied attributes and methods to reflect the current 
473          data model. 
474   
475      .. versionchanged:: 0.6.1 
476          The class can be instantiated with a callable.
