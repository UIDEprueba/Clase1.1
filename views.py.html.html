31   <span class="s0">class </span><span class="s1">View:</span>
32       <span class="s3">&quot;&quot;&quot;Subclass this class and override :meth:`dispatch_request` to 
33       create a generic class-based view. Call :meth:`as_view` to create a 
34       view function that creates an instance of the class with the given 
35       arguments and calls its ``dispatch_request`` method with any URL 
36       variables. 
37    
38       See :doc:`views` for a detailed guide. 
39    
40       .. code-block:: python 
41    
42           class Hello(View): 
43               init_every_request = False 
44    
45               def dispatch_request(self, name): 
46                   return f&quot;Hello, {name}!&quot; 
47    
48           app.add_url_rule( 
49               &quot;/hello/&lt;name&gt;&quot;, view_func=Hello.as_view(&quot;hello&quot;) 
50           ) 
51    
52       Set :attr:`methods` on the class to change what methods the view 
53       accepts. 
54    
55       Set :attr:`decorators` on the class to apply a list of decorators to 
56       the generated view function. Decorators applied to the class itself 
57       will not be applied to the generated view function! 
58    
59       Set :attr:`init_every_request` to ``False`` for efficiency, unless 
60       you need to store request-global data on ``self``. 
61       &quot;&quot;&quot;</span>

154      <span class="s3">&quot;&quot;&quot;Dispatches request methods to the corresponding instance methods. 
155      For example, if you implement a ``get`` method, it will be used to 
156      handle ``GET`` requests. 
157   
158      This can be useful for defining a REST API. 
159   
160      :attr:`methods` is automatically set based on the methods defined on 
161      the class. 
162   
163      See :doc:`views` for a detailed guide. 
164   
165      .. code-block:: python 
166   
167          class CounterAPI(MethodView): 
168              def get(self): 
169                  return str(session.get(&quot;counter&quot;, 0)) 
170   
171              def post(self): 
172                  session[&quot;counter&quot;] = session.get(&quot;counter&quot;, 0) + 1 
173                  return redirect(url_for(&quot;counter&quot;))

176              &quot;/counter&quot;, view_func=CounterAPI.as_view(&quot;counter&quot;)
