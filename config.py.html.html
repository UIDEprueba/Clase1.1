22   Implements the Distutils 'config' command, a (mostly) empty command class 
23   that exists mainly to be sub-classed by specific module distributions and 
24   applications.  The idea is that while every &quot;config&quot; command is different, 
25   at least they're all named the same, and users always see &quot;config&quot; in the 
26   list of standard commands.  Also, this is a good place to put common 
27   configure-like tasks: &quot;try to compile this C code&quot;, or &quot;figure out where 
28   this header file lives&quot;.
