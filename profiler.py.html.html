22   Application Profiler 
23   ==================== 
24    
25   This module provides a middleware that profiles each request with the 
26   :mod:`cProfile` module. This can help identify bottlenecks in your code 
27   that may be slowing down your application.

51   <span class="s2">class </span><span class="s1">ProfilerMiddleware:</span>
52       <span class="s0">&quot;&quot;&quot;Wrap a WSGI application and profile the execution of each 
53       request. Responses are buffered so that timings are more exact. 
54    
55       If ``stream`` is given, :class:`pstats.Stats` are written to it 
56       after each request. If ``profile_dir`` is given, :mod:`cProfile` 
57       data files are saved to that directory, one file per request. 
58    
59       The filename can be customized by passing ``filename_format``. If 
60       it is a string, it will be formatted using :meth:`str.format` with 
61       the following fields available: 
62    
63       -   ``{method}`` - The request method; GET, POST, etc. 
64       -   ``{path}`` - The request path or 'root' should one not exist. 
65       -   ``{elapsed}`` - The elapsed time of the request. 
66       -   ``{time}`` - The time of the request. 
67    
68       If it is a callable, it will be called with the WSGI ``environ`` 
69       dict and should return a filename. 
70    
71       :param app: The WSGI application to wrap. 
72       :param stream: Write stats to this stream. Disable with ``None``. 
73       :param sort_by: A tuple of columns to sort stats by. See 
74           :meth:`pstats.Stats.sort_stats`. 
75       :param restrictions: A tuple of restrictions to filter stats by. See 
76           :meth:`pstats.Stats.print_stats`. 
77       :param profile_dir: Save profile data files to this directory. 
78       :param filename_format: Format string for profile data file names, 
79           or a callable returning a name. See explanation above. 
80    
81       .. code-block:: python 
82    
83           from werkzeug.middleware.profiler import ProfilerMiddleware 
84           app = ProfilerMiddleware(app) 
85    
86       .. versionchanged:: 0.15 
87           Stats are written even if ``profile_dir`` is given, and can be 
88           disable by passing ``stream=None``. 
89    
90       .. versionadded:: 0.15 
91           Added ``filename_format``. 
92    
93       .. versionadded:: 0.9 
94           Added ``restrictions`` and ``profile_dir``.
