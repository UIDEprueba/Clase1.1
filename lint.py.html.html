229  <span class="s2">class </span><span class="s1">LintMiddleware:</span>
230      <span class="s0">&quot;&quot;&quot;Warns about common errors in the WSGI and HTTP behavior of the 
231      server and wrapped application. Some of the issues it checks are: 
232   
233      -   invalid status codes 
234      -   non-bytes sent to the WSGI server 
235      -   strings returned from the WSGI application 
236      -   non-empty conditional responses 
237      -   unquoted etags 
238      -   relative URLs in the Location header 
239      -   unsafe calls to wsgi.input 
240      -   unclosed iterators 
241   
242      Error information is emitted using the :mod:`warnings` module. 
243   
244      :param app: The WSGI application to wrap. 
245   
246      .. code-block:: python 
247   
248          from werkzeug.middleware.lint import LintMiddleware 
249          app = LintMiddleware(app) 
250      &quot;&quot;&quot;</span>
