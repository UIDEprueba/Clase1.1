441      <span class="s4">&quot;&quot;&quot; 
442      Behaves just like :class:`PoolManager`, but sends all requests through 
443      the defined proxy, using the CONNECT method for HTTPS URLs. 
444   
445      :param proxy_url: 
446          The URL of the proxy to be used. 
447   
448      :param proxy_headers: 
449          A dictionary containing headers that will be sent to the proxy. In case 
450          of HTTP they are being sent with each request, while in the

453
