21   X-Forwarded-For Proxy Fix 
22   ========================= 
23    
24   This module provides a middleware that adjusts the WSGI environ based on 
25   ``X-Forwarded-`` headers that proxies in front of an application may 
26   set. 
27    
28   When an application is running behind a proxy server, WSGI may see the 
29   request as coming from that server rather than the real client. Proxies 
30   set various headers to track where the request actually came from. 
31    
32   This middleware should only be used if the application is actually 
33   behind such a proxy, and should be configured with the number of proxies 
34   that are chained in front of it. Not all proxies set all the headers. 
35   Since incoming headers can be faked, you must set how many proxies are 
36   setting each header so the middleware knows what to trust. 
37

53   <span class="s2">class </span><span class="s1">ProxyFix:</span>
54       <span class="s0">&quot;&quot;&quot;Adjust the WSGI environ based on ``X-Forwarded-`` that proxies in 
55       front of the application may set. 
56    
57       -   ``X-Forwarded-For`` sets ``REMOTE_ADDR``. 
58       -   ``X-Forwarded-Proto`` sets ``wsgi.url_scheme``. 
59       -   ``X-Forwarded-Host`` sets ``HTTP_HOST``, ``SERVER_NAME``, and 
60           ``SERVER_PORT``. 
61       -   ``X-Forwarded-Port`` sets ``HTTP_HOST`` and ``SERVER_PORT``. 
62       -   ``X-Forwarded-Prefix`` sets ``SCRIPT_NAME``. 
63    
64       You must tell the middleware how many proxies set each header so it 
65       knows what values to trust. It is a security issue to trust values 
66       that came from the client rather than a proxy. 
67    
68       The original values of the headers are stored in the WSGI 
69       environ as ``werkzeug.proxy_fix.orig``, a dict. 
70    
71       :param app: The WSGI application to wrap. 
72       :param x_for: Number of values to trust for ``X-Forwarded-For``. 
73       :param x_proto: Number of values to trust for ``X-Forwarded-Proto``. 
74       :param x_host: Number of values to trust for ``X-Forwarded-Host``. 
75       :param x_port: Number of values to trust for ``X-Forwarded-Port``. 
76       :param x_prefix: Number of values to trust for 
77           ``X-Forwarded-Prefix``. 
78    
79       .. code-block:: python 
80    
81           from werkzeug.middleware.proxy_fix import ProxyFix 
82           # App is behind one proxy that sets the -For and -Host headers. 
83           app = ProxyFix(app, x_for=1, x_host=1)
