140  <span class="s0">class </span><span class="s1">Blueprint(Scaffold):</span>
141      <span class="s4">&quot;&quot;&quot;Represents a blueprint, a collection of routes and other 
142      app-related functions that can be registered on a real application 
143      later. 
144   
145      A blueprint is an object that allows defining application functions 
146      without requiring an application object ahead of time. It uses the 
147      same decorators as :class:`~flask.Flask`, but defers the need for an 
148      application by recording them for later registration. 
149   
150      Decorating a function with a blueprint creates a deferred function 
151      that is called with :class:`~flask.blueprints.BlueprintSetupState` 
152      when the blueprint is registered on an application. 
153   
154      See :doc:`/blueprints` for more information. 
155   
156      :param name: The name of the blueprint. Will be prepended to each 
157          endpoint name. 
158      :param import_name: The name of the blueprint package, usually 
159          ``__name__``. This helps locate the ``root_path`` for the 
160          blueprint. 
161      :param static_folder: A folder with static files that should be 
162          served by the blueprint's static route. The path is relative to 
163          the blueprint's root path. Blueprint static files are disabled 
164          by default. 
165      :param static_url_path: The url to serve static files from. 
166          Defaults to ``static_folder``. If the blueprint does not have 
167          a ``url_prefix``, the app's static route will take precedence, 
168          and the blueprint's static files won't be accessible. 
169      :param template_folder: A folder with templates that should be added 
170          to the app's template search path. The path is relative to the 
171          blueprint's root path. Blueprint templates are disabled by 
172          default. Blueprint templates have a lower precedence than those 
173          in the app's templates folder. 
174      :param url_prefix: A path to prepend to all of the blueprint's URLs, 
175          to make them distinct from the rest of the app's routes. 
176      :param subdomain: A subdomain that blueprint routes will match on by 
177          default. 
178      :param url_defaults: A dict of default values that blueprint routes 
179          will receive by default. 
180      :param root_path: By default, the blueprint will automatically set 
181          this based on ``import_name``. In certain situations this 
182          automatic detection can fail, so the path can be specified 
183          manually instead. 
184   
185      .. versionchanged:: 1.1.0 
186          Blueprints have a ``cli`` group to register nested CLI commands. 
187          The ``cli_group`` parameter controls the name of the group under 
188          the ``flask`` command. 
189   
190      .. versionadded:: 0.7
