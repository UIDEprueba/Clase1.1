182  <span class="s0">def </span><span class="s1">make_response(*args: t.Any) -&gt; </span><span class="s4">&quot;Response&quot;</span><span class="s1">:</span>
183      <span class="s3">&quot;&quot;&quot;Sometimes it is necessary to set additional headers in a view.  Because 
184      views do not have to return response objects but can return a value that 
185      is converted into a response object by Flask itself, it becomes tricky to 
186      add headers to it.  This function can be called instead of using a return 
187      and you will get a response object which you can use to attach headers. 
188   
189      If view looked like this and you want to add a new header:: 
190   
191          def index(): 
192              return render_template('index.html', foo=42) 
193   
194      You can now do something like this:: 
195   
196          def index(): 
197              response = make_response(render_template('index.html', foo=42)) 
198              response.headers['X-Parachutes'] = 'parachutes are cool' 
199              return response 
200   
201      This function accepts the very same arguments you can return from a 
202      view function.  This for example creates a response with a 404 error 
203      code:: 
204   
205          response = make_response(render_template('not_found.html'), 404) 
206   
207      The other use case of this function is to force the return value of a 
208      view function into a response which is helpful with view 
209      decorators:: 
210   
211          response = make_response(view_function()) 
212          response.headers['X-Parachutes'] = 'parachutes are cool' 
213   
214      Internally this function does the following things: 
215   
216      -   if no arguments are passed, it creates a new response argument 
217      -   if one argument is passed, :meth:`flask.Flask.make_response` 
218          is invoked with it. 
219      -   if more than one argument is passed, the arguments are passed 
220          to the :meth:`flask.Flask.make_response` function as tuple. 
221   
222      .. versionadded:: 0.6 
223      &quot;&quot;&quot;</span>
224      <span class="s0">if not </span><span class="s1">args:</span>
225          <span class="s0">return </span><span class="s1">current_app.response_class()</span>
226      <span class="s0">if </span><span class="s1">len(args) == </span><span class="s5">1</span><span class="s1">:</span>
227          <span class="s1">args = args[</span><span class="s5">0</span><span class="s1">]</span>
228      <span class="s0">return </span><span class="s1">current_app.make_response(args)  </span><span class="s2"># type: ignore</span>
